# UI Library Reference

A comprehensive catalog of UI component libraries, icon sets, animation libraries, styling frameworks, and visual/glass/neumorphism tools.

---

## UI Component Libraries (React)

| Library | Category | Description |
|---|---|---|
| [AlignUI](https://www.alignui.com/) | UI components + blocks | Tailwind-styled React components/blocks split between free base components and paid Pro blocks/templates with a perpetual license. |
| [Ant Design](https://ant.design/) | UI components | Enterprise-oriented React design system with a large component surface area (tables/forms/layout) and strong defaults for business apps. |
| [Chakra UI](https://chakra-ui.com/) | UI components | Accessible, composable React components with style props and theming; good for rapid product UI with consistent spacing/typography tokens. |
| [Float UI](https://floatui.com/) | Tailwind components/templates | Tailwind components + templates explicitly targeting fast landing pages/app shells. No redistribution as a component library. |
| [HeroUI (NextUI)](https://www.heroui.com/) | UI components | Modern React component library focused on accessible, themeable components with fast DX (Tailwind-friendly, slot-based customization, CLI tooling). |
| [Kibo UI](https://www.kiboui.com/) | shadcn registry | Component/blocks registry designed to drop into shadcn/ui projects (Tailwind vars, Radix-friendly), aimed at extending the catalog quickly. |
| [Mantine](https://mantine.dev/) | UI components + hooks | Large React component suite plus utility hooks (forms, modals, notifications, etc.). Strong out-of-box theming and DX. |
| [MUI (Material UI)](https://mui.com/) | UI components | Material Design-based React components with strong ecosystem; common choice for dashboards and B2B apps. MUI X adds paid Pro/Premium data grid and chart features. |
| [Preline](https://preline.co/) | Tailwind components | Tailwind component library + examples/blocks with a dual-license scheme and optional Pro offering. |
| [PrimeReact](https://primereact.org/) | UI components | Big UI suite (tables, charts integrations, complex inputs). Useful for dashboard-heavy apps; lots of components and themes. |
| [React Bootstrap](https://react-bootstrap.netlify.app/) | UI components | Bootstrap components rebuilt as real React components (no jQuery), for teams standardizing on Bootstrap styling. |
| [Relume](https://www.relume.io/) | Component library + tooling | Subscription product bundling a large component library (React/Tailwind + Webflow + Figma workflows). License limits redistribution/competing libraries. |
| [Reshaped](https://reshaped.so/) | UI system | Design-system approach with React components + Figma libraries + theming tools, geared toward keeping design/code aligned. |
| [Semantic UI React](https://react.semantic-ui.com/) | UI components | React bindings for Semantic UI styling; quick to assemble UIs if you're already aligned with Semantic's CSS approach. |
| [Sera UI](https://seraui.com/) | UI components (React/Next) | Free React and Next.js component library with animated components: marquee scrollers, orbit animations, glitch effects, network visualizations, glow lines, and more. |
| [shadcn/ui](https://ui.shadcn.com/) | UI components (copy/paste) | Copy/paste components built on Radix primitives + Tailwind. You own the code; great for custom design systems and long-term maintenance. |
| [Untitled UI](https://www.untitledui.com/) | UI components | Large collection of Tailwind + React Aria components. Many components are OSS; PRO adds more advanced pieces. |

## Tailwind Plugins & Utility Libraries

| Library | Category | Description |
|---|---|---|
| [daisyUI](https://daisyui.com/) | Tailwind component plugin | Tailwind plugin that adds semantic component classes (btn/card/modal/etc.) with themes; quick UI without writing many utilities. |
| [Flowbite](https://flowbite.com/) | Tailwind components | Tailwind component library with strong documentation and broad coverage. Easy to apply glass effects via backdrop-blur and opacity utilities. |
| [FlyonUI](https://flyonui.com/) | Tailwind components | Open-source Tailwind component library and blocks; good for shipping quickly, with optional Pro upgrades. |
| [Tailwind CSS](https://tailwindcss.com/) | Styling framework | Utility-first CSS framework (design tokens via config, responsive variants, arbitrary values). Common base layer for shadcn-style ecosystems. |

## Headless / Accessibility Primitives

| Library | Category | Description |
|---|---|---|
| [Base UI](https://base-ui.com/) | Headless / unstyled | Unstyled React components emphasizing accessibility + composability; designed as a foundational layer you fully style yourself. |
| [Headless UI](https://headlessui.com/) | Headless / accessibility | Unstyled, fully accessible React components (menus, dialogs, disclosures). You bring the styling — often Tailwind. |
| [Radix UI](https://www.radix-ui.com/) | Headless primitives | High-quality accessible primitives widely used underneath shadcn/ui; great base for custom components. |
| [React Aria](https://react-spectrum.adobe.com/react-aria/) | Headless / a11y hooks | Low-level hooks + patterns for WAI-ARIA compliant components; designed for building your own UI while keeping accessibility solid. |

| [SpecularCSS](https://github.com/lvl99/specularcss) | CSS (skeuo/material) | Open-source CSS rules for adding material/specular styling (gloss, depth, highlight effects). Not React-specific; works with any markup/CSS pipeline. |
| [Vizor](https://github.com/leoafarias/vizor) | UI kit (Flutter) | Sci-fi/cyberpunk UI kit for Flutter. Not React/Vue — a different ecosystem, but a strong reference for the aesthetic. |

## Retro / Pixel / Nostalgic UI

| Library | Category | Description |
|---|---|---|
| [7.css](https://github.com/khang-nd/7.css) | CSS framework (Windows 7) | JS-independent, tree-shakeable CSS framework for building faithful recreations of the Windows 7 UI. Extends the 98.css family to the Aero era. |
| [98.css](https://jdan.github.io/98.css/) | CSS framework (Windows 98) | CSS-only library for building faithful recreations of Windows 98 UIs. Drop in the stylesheet and use semantic HTML to get pixel-perfect Win98 components. |
| [8bitcn/ui](https://www.8bitcn.com/) | UI components (React) | 8-bit styled component set and code distribution platform. Copy/paste retro pixel components (buttons, dropdowns, windows, quest logs) into your favorite framework. Open source. |
| [a2000 (a2k)](https://a2000-docs.netlify.app/) | UI library (retro web) | UI library recreating the feel of the early 2000s web using modern tools. Covers windows, panels, taskbars, progress bars, marquees, and more. Work in progress. |
| [Aero UI](https://github.com/DanielAraldi/aero-ui) | UI components (Tailwind) | Tailwind-based component library evoking lightness, fluidity, and sophistication — inspired by Windows Aero's glass and translucency aesthetic. |
| [Arcade UI](https://arcade-ui.pages.dev/) | UI components (React) | Arcade-themed React component library with a retro gaming visual style — chunky pixel-inspired controls and layouts for game-adjacent UIs. |
| [BOOTSTRA.386](https://github.com/kristopolous/BOOTSTRA.386) | CSS theme (Bootstrap) | Vintage 1980s DOS-inspired Twitter Bootstrap theme. Monochrome green-on-black terminal aesthetic layered on top of Bootstrap's grid and components. |
| [classic.css](https://github.com/npjg/classic.css) | CSS framework (Classic Mac) | Generates a Classic Mac interface in the browser. CSS-only recreation of early Macintosh OS UI chrome — windows, menus, buttons, and more. |
| [NES.css](https://nostalgic-css.github.io/NES.css/) | CSS framework (NES 8-bit) | NES-style (8-bit) CSS framework. Provides pixel-bordered components styled after the NES console aesthetic. No layout opinions — pairs well with React, Vue, or Tailwind. |
| [nes-ui-react](https://kyr0.github.io/nes-ui-react/) | UI components (React) | Comprehensive NES-style React component library forked and fully refactored from NES.css. Adds modern Sass, TypeScript, dark/light mode, NTSC color palette, and many new components. |
| [Pixelact UI](https://www.pixelactui.com/) | UI components (React) | Pixel-art flavored React component library for building interfaces with a retro pixel aesthetic. |
| [PSone.css](https://micah5.github.io/PSone.css/) | CSS framework (PS1) | PlayStation 1-style CSS framework inspired by NES.css. Recreates PS1-era UI elements: buttons, containers, progress bars, form inputs, and controller-button icons. |
| [Pxlkit](https://pxlkit.xyz/) | UI kit + icons (React) | Open-source retro React UI kit bundled with 211+ pixel art icons across 6 themed packs (gamification, feedback, social, weather, UI, effects). 53 components with a TypeScript-first API. |
| [react-windows-xp](https://github.com/luiz-chagas/react-windows-xp) | UI components (React) | Windows XP-like React components; renders classic Luna-themed windows, buttons, and chrome elements straight in the browser. |
| [retro-react](https://github.com/retro-react/retro-react) | UI components (React) | React UI library styled after 90s-era websites — scanlines, pixelated fonts, loud colors, and period-accurate controls. |
| [retro-ui](https://github.com/sombreroEnPuntas/retro-ui) | UI components (React) | NES-style 8-bit React component library built with styled-components. Pixel-bordered controls in the NES.css aesthetic, packaged for React. |
| [RetroUI](https://www.retroui.dev/) | UI components (React) | Neo-brutalism styled React component library. Bold borders, offset shadows, and a raw chunky aesthetic — perfect for landing pages with a loud, retro-brutalist look. |
| [SNES.css](https://snes-css.sadlative.com/) | CSS framework (SNES 16-bit) | CSS framework inspired by 90s 16-bit SNES-era video games. Pixel art style with themed text palettes and UI components. No JS or layout styles — works alongside React or Tailwind. |
| [system.css](https://github.com/sakofchit/system.css) | CSS framework (retro Apple) | Design system for building retro Apple-style interfaces — recreates the look of classic Mac OS (System 6/7 era) using CSS only. Framework-agnostic. |
| [TheSims.css](https://github.com/inbn/TheSims.css) | CSS framework (The Sims) | A "design system" based on the 2000 life simulation game The Sims. Recreates the game's distinctive UI chrome in pure CSS. |
| [XP.css](https://botoxparty.github.io/XP.css/) | CSS framework (Windows XP) | Built on top of 98.css; adds Windows XP Luna theme support alongside Win98. CSS-only, no JavaScript — just classes on standard HTML. |
